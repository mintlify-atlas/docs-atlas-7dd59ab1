## ¿Qué es este proyecto?

El sitio web de la IEM Emilio Cifuentes fue rediseñado desde cero para resolver problemas de usabilidad, rendimiento y adaptación a dispositivos móviles detectados en el sitio anterior. El resultado es una interfaz moderna, completamente responsive, construida únicamente con **HTML5** y **CSS3** — sin frameworks ni dependencias externas.

<Steps>
  <Step title="Clona el repositorio">
    Descarga el código fuente desde GitHub para empezar a explorar o contribuir.
  </Step>
  <Step title="Abre index.html en tu navegador">
    No se requiere servidor ni proceso de build. Abre el archivo directamente en cualquier navegador moderno.
  </Step>
  <Step title="Explora las páginas">
    Navega por las cinco secciones del sitio: Inicio, Nosotros, Noticias, Recursos y Contacto.
  </Step>
  <Step title="Contribuye siguiendo las convenciones">
    Usa la guía de commits y el estándar de comentarios para mantener la calidad del código.
  </Step>
</Steps>

<Note>
  El proyecto está en fase **MVP**. Algunas funcionalidades como el buscador, el calendario académico y los detalles de noticias individuales están marcadas como pendientes para versiones futuras.
</Note>
